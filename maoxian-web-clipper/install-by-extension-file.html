Using extension file to install MaoXian Web Clipper

简体

Version info

  • Current version: 0.7.70

Download software package

The belowing files that ends with checksum.txt are used to verify that the downloaded package are completed and not modified.

Target browser is Chromium based browser

CRX package

After you’ve installed this package, the browser can update MaoXian automatically. Note that this package is not suit for Windows + Chrome or MacOS + Chrome. If you are one of these environment, see the blowing “Installation” section.

Note: If the browser that you currently use is a Chromium based browser, such as Chrome, Chromium, Edge etc. Using belowing links, and after you complete the download, rename it by removing the .zip part. The reason of doing this is that these browsers will try to install the CRX file and if the CRX file isn’t came from Chrome Web Store it’ll get deleted. So we use this file extension trick to walk around.

  • maoxian-web-clipper-chromium-0.7.70.crx.zip
  • maoxian-web-clipper-chromium-0.7.70.crx.zip.checksum.txt

Note: If the browser that you currently use is not a Chromium based browser, using belowing links:

  • maoxian-web-clipper-chromium-0.7.70.crx
  • maoxian-web-clipper-chromium-0.7.70.crx.checksum.txt

ZIP package

After you’ve install this package, the browser can’t update MaoXian automatically.

  • maoxian-web-clipper-chromium-0.7.70.zip
  • maoxian-web-clipper-chromium-0.7.70.zip.checksum.txt

Target browser is Firefox

If you choose to install zip package, Firefox won’t update MaoXian automatically. It’s better to install MaoXian on Firefox through AMO

  • maoxian-web-clipper-firefox-0.7.70.zip
  • maoxian-web-clipper-firefox-0.7.70.zip.checksum.txt

Verifing the checksum

Terminal uses can use belowing commands to calculate the checksum and compare the result with the checksum.txt file. If the checksum matchs, it proves the package is completed and not modified.

# use md5 algorithm to calculate the checksum
$ md5sum maoxian-web-clipper-x.x.x.zip

# use sha256 algorithm to calculate the checksum
$ sha256sum maoxian-web-clipper-x.x.x.zip

# use sha512 algorithm to calculate the checksum
$ sha512sum maoxian-web-clipper-x.x.x.zip

Installing

Install MaoXian on Chromium

If your environment is Windows + Chrome or MacOS + Chrome, Chrome will disable and won’t allow enabling any browser extension that was not installed from Chrome Web Store. This “for your security” policy will take away your ability of installing any browser extension that is distributed through other methods. Chrome don’t respect both users and developers, they just want to gain control. hence we highly recomend you leave Chrome for other better browsers (such as: Chromium , ungoogled-chromium or other Chromium-based browsers).

Install by drag and drop the crx package

This is the most recommended installing method. The packages that are installed using this method, can update automatically in the future.

  1. download the crx package
  2. open the extension manage page on Chromium (by enter the about:extensions URL)
  3. turn Developer mode on
  4. drag the downloaded crx package to the extension manage page and drop to install.

Note that if the 4th step can’t trigger the installation, restart your browser with keeping Developer mode on. and try again.

Install by load unpacked zip package

  1. download the zip package.
  2. unzip the package. Note that the package were packed without the root directory. So to avoid messing current directory, move the zip package to a new empty directory and unzip it. If you can’t unzip it successfully, try another unzip software.
  3. open the extension manage page on Chromium (by enter the about:extensions URL)
  4. turn Developer mode on
  5. click the Load unpacked button and choose the directory that you just unziped.

warning: The packages that are installed using this method will not update automatically.

Install MaoXian on Firefox (using zip package)

The most convenient method to install MaoXian on Firefox is through AMO, Here’s the link.

Currently, Only the ESR version and Beta version of Firefox can use this installing method.

  1. turn zip installing on. enter the about:config URL, search xpinstall.signatures.required and change it’s value to false.
  2. enter the about:addons URL, click the Extensions menu on the left, click the dropdown button which with a cog icon, click the Install Add-on From File menu item and select the downloaded zip package.

warning: The packages that are installed using this method will not update automatically.
