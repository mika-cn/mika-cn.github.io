Check this before sending feedback

To avoid duplicate reporting the same problem again and again, please check below pages first:

  • Frequently asked questions page
  • Search keywords on issues page

Please post all the details

When you describe your problem, more details more helpful. So please provide belowing infos:

  • The current URL of web page (If you encountered a clipping bug)
  • Your software environment: OS type and version, Browser type and version, MaoXian version, NativeApp version (If you have installed)
  • Your configuration of MaoXian (export it on settings > reset and backup )
  • Please describe all you details about the problem
    • such as:
      • Is this happened everytime or ramdomly?
      • Is this happened only on this web page or all the pages on this website or every web page?
      • etc.
