简介

这是一个离线的 HTML 网页，它让你可以像扩展的历史页面那样，列出所有的裁剪信息 并且 支持搜索。下载这个页面并且配置好扩展后，扩展就能及时更新该页面的裁剪信息。这意味着你可以随时就把 mx-wc 文件夹同步或者拷贝到另一台设备，再用浏览器打开这个「离线索引页面」，就能浏览和搜索你所有的裁剪信息。另，你也可以把这个页面部署到你自己的服务器上，或者是 Github pages，以及任何支持发布网页的服务。

顺带唠叨一句：如果你会 Web(HTML, CSS, javascript) 开发，你完全可以基于这个页面，修改其成你喜欢的样式，甚至添加功能。

当前版本: 0.0.5

使用方法

先下载该页面 && 解压它

温馨提示: MaoXian 扩展从 0.1.34 版本开始，支持你配置根目录的名称，该索引页面也从 0.0.4 版本开始，多了一个配置文件 (config.js) ，如果你修改了扩展设置页面里的根目录，你也要修改这个配置文件里面的根目录项（rootFolder）。下文中提到的 mx-wc 即是默认的根目录，你的可能由于改了配置，而不同。

致歉：本次 MaoXian 扩展的更新（0.1.46），无法兼容老版本的「离线索引页」，所以当扩展更新后，使用「离线索引页」的老用户，需要升级到新版本，才能正常使用。此实属无奈之举，造成诸多不便，非常抱歉。

步骤 1: 移动解压出来的文件夹

把解压出来的文件夹(默认叫history)，移动到扩展裁剪的根目录（$download/mx-wc), 移动后的路径 $download/mx-wc/history.

目录名 “history” 是默认的，你可以重命名它，甚至把目录里的文件移动到 $download/mx-wc 目录下面。更多细节，请看下面

假设你的 $download 目录是 /home/jack/Downloads, 那么你的 $download/mx-wc 目录即是 /home/jack/Downloads/mx-wc.

离线页面路径 是否支持 描述
/home/jack/Downloads/mx-wc/offline-index/index.html 是 我们把 history 重命名为 offline-index
/home/jack/Downloads/mx-wc/offline-index.html 是 我们把 history 目录下的所有文件移动到 $download/mx-wc 目录下并且把 index.html 重命名为 offline-index.html
/home/jack/Downloads/mx-wc/offline-pages/history/index.html 是 我们把这个页面放在了更深的目录下
/home/jack/history/index.html 否 这个页面不能放到 $download/mx-wc 目录之外

步骤 2: 用浏览器打开它

使用你喜欢的浏览器，打开 index.html (即$download/mx-wc/history/index.html), 该页面即是我们所说的「离线索引页面」

步骤 3: 配置 MaoXian 扩展

当你做完步骤2,你应该能看到一个和扩展的历史页面很像的网页。该网页上会有两条裁剪信息，是样本。这两条信息存储在一个叫 clippings.js (即 $download/mx-wc/history/clippings.js) 里面。第三步是到扩展的设置页面，配置「离线索引页面」的相关配置。

具体是这样配置:

  • 配置脚本文件的路径，即clippings.js 的路径。默认是 $MX-WC/history/clippings.js, 如果你没有改过解压出来的文件夹的名字。这个配置则无需修改
  • 勾选自动生成脚本文件 (让扩展及时地更新这个脚本文件，时时保持最新)
  • 页面上有个「马上生成」按钮，你现在就可以点一下看看，然后再刷新刚才你打开的 index.html , 如果页面上的记录不再是那两条样本信息，恭喜你，配置成功了。
