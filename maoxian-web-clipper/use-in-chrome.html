Use in chrome / chromium

1. Download crx file below

mx-web-clipper-chrome.crx

2. Open Chrome extension page: about:extensions

3. method1: drag crx file directly

Drag downloaded crx file to chrome extension page.

3. method2: use developer mode (if method1 not work)

  • rename crx file to .zip, and extract file (if your system is windows, don’t use builtin extract software)

  • open developer mode
  • load unpacked, choose fold you just extract.
